Tag Class

O atributo HTML classé usado para especificar uma classe para um elemento HTML. Vários elementos HTML podem compartilhar a mesma classe.

Usando o atributo class

O atributo class geralmente é usado para apontar para um nome de classe em uma folha de estilo. Também pode ser usado por JavaScript para acessar e manipular elementos com o nome de classe específico. No exemplo a seguir temos três div elementos com um atributo class com o valor "cidade". Dica: O classatributo pode ser usado em qualquer elemento HTML.
Observação: o nome da classe diferencia maiúsculas de minúsculas!

Atributo de identificação - id- HTML

O atributo HTML id é usado para especificar um id exclusivo para um elemento HTML.
Você não pode ter mais de um elemento com o mesmo id em um documento HTML.

Observação: o nome do id diferencia maiúsculas de minúsculas!
Nota: O nome id deve conter pelo menos um caractere, não pode começar com um número e não deve conter espaços em branco (espaços, tabulações, etc.).
