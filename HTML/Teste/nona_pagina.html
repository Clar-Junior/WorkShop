Elementos e técnicas de layout HTML

Elementos de Esquema HTML:

tag descrição
header Define um cabeçalho para um documento ou uma seção
nav Define um conjunto de links de navegação
section Define uma seção em um documento
article Define um conteúdo independente e independente
aside Define o conteúdo além do conteúdo (como uma barra lateral)
footer Define um rodapé para um documento ou uma seção
details Define detalhes adicionais que o usuário pode abrir e fechar sob demanda

Técnicas de layout HTML

Existem quatro técnicas diferentes para criar layouts de várias colunas. Cada técnica tem seus prós e contras:

  • estrutura CSS
    • Se você deseja criar seu layout rapidamente, pode usar um framework CSS, como W3.CSS ou Bootstrap .

  • Propriedade flutuante do CSS
    • É comum fazer layouts de web inteiros usando a floatpropriedade CSS. Float é fácil de aprender - você só precisa se lembrar de como as propriedades floate clearfuncionam. Desvantagens: Os elementos flutuantes ficam presos ao fluxo do documento, o que pode prejudicar a flexibilidade.

  • CSS flexbox
    • O uso do flexbox garante que os elementos se comportem de maneira previsível quando o layout da página deve acomodar diferentes tamanhos de tela e diferentes dispositivos de exibição.

  • Grade CSS
    • O Módulo de layout de grade CSS oferece um sistema de layout baseado em grade, com linhas e colunas, facilitando o design de páginas da Web sem a necessidade de usar flutuações e posicionamento.
